# 💸 Expense Tracker Frontend

A responsive Expense Tracker built with **React** and **Vite** that allows users to add, view, and delete expenses through a RESTful backend API.

## 🚀 Tech Stack

- React
- Vite
- Axios
- CSS

## ✨ Features

- Add new expenses
- View all expenses
- Delete expenses
- Total expense summary
- Responsive UI

## 📂 Folder Structure

```
src/
├── components/
├── services/
├── App.jsx
├── App.css
├── index.css
└── main.jsx
```

## ⚙️ Installation

```

```

```javascript

```
